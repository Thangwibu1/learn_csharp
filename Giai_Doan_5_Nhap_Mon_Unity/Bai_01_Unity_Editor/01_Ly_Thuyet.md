# Bài 1: Unity Editor

## Mục tiêu

- Biết các cửa sổ chính của Unity
- Hiểu Scene, Game, Hierarchy, Inspector, Project

## Unity Editor là gì?

Unity Editor là môi trường làm việc nơi bạn tạo game.

Các cửa sổ quan trọng:

- `Hierarchy`: danh sách object trong scene
- `Scene`: nơi bạn sắp xếp thế giới game
- `Game`: góc nhìn khi game chạy
- `Inspector`: nơi chỉnh thông tin object
- `Project`: nơi chứa file asset và script

## Tại sao phải dùng cái này?

Vì trước khi code, bạn phải biết mình đang thao tác ở đâu trong Unity.

## Lỗi sai thường gặp của người mới

- Không phân biệt `Scene` và `Game`
- Sửa nhầm object khác trong `Hierarchy`
- Không hiểu Inspector đang hiển thị cái gì

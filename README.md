# Khoa Hoc C# va Unity Tu So 0 den Lam Game

Day la bo tai lieu hoc C# va Unity duoc sap xep theo lo trinh 7 giai doan, viet cho nguoi moi hoan toan.

Muc tieu cua khoa hoc:

- Hoc C# tu nen tang co ban den nang cao.
- Xay nen tu duy OOP dung cach.
- Hieu collections, data structures va game architecture.
- Lam quen voi Unity Editor va Unity scripting.
- Di den cac bai thuc chien nhu state machine, observer pattern, object pooling va save system.

## Cach hoc de dat hieu qua cao nhat

1. Hoc theo dung thu tu giai doan.
2. O moi bai, doc `01_Ly_Thuyet.md` truoc.
3. Sau do mo `02_Thuc_Hanh.cs` va tu go lai code.
4. Tu giai `03_Bai_Tap.md` truoc khi xem lai ly thuyet.
5. Neu bai co file `04_Mo_Rong_Chi_Tiet.md`, hay doc tiep de dao sau hon.

## Trong khoa hoc nay co gi dac biet?

- Giai thich theo kieu su pham cho nguoi moi.
- Luon co phan tai sao phai dung cai nay.
- Luon co loi sai thuong gap cua nguoi moi.
- Co nhieu vi du game gan voi Unity.
- Co cac chu de can thiet cho lap trinh game that:
  - Inheritance vs Composition
  - Interface de giam phu thuoc
  - Generic Class<T>
  - Singleton dung an toan
  - List, Dictionary, Queue, Stack trong game
  - State Machine cho AI quai
  - Observer Pattern bang Event/Action

# Bài 1: UI và Animation

## Mục tiêu

- Biết UI cơ bản trong Unity
- Hiểu Animation giúp game sống động hơn

## UI là gì?

UI là giao diện mà người chơi nhìn thấy:

- máu
- mana
- nút bấm
- inventory
- pause menu

## Animation là gì?

Animation làm object chuyển động mượt và có cảm giác sống.

## Tại sao phải dùng cái này?

Game không chỉ cần chạy đúng logic, nó còn cần dễ hiểu và có cảm xúc.

## Lỗi sai thường gặp của người mới

- UI cập nhật bằng `Update()` dù dữ liệu hiếm khi đổi
- Animation và logic gameplay phụ thuộc lẫn nhau quá chặt

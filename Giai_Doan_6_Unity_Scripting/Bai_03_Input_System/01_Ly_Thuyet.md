# Bài 3: Input System

## Mục tiêu

- Hiểu input là gì
- Biết đọc input cơ bản

## Input là gì?

Input là dữ liệu từ người chơi gửi vào game.

Ví dụ:

- bấm phím
- chạm màn hình
- click chuột

## Tại sao phải dùng cái này?

Vì game không thể chơi được nếu không nhận lệnh từ người dùng.

## Lỗi sai thường gặp của người mới

- Viết input dính chặt vào mọi hệ thống
- Không tách input và movement khi dự án lớn lên

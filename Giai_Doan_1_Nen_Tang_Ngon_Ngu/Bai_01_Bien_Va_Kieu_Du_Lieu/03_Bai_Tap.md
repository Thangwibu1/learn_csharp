# Bài tập Bài 1: Biến và Kiểu Dữ Liệu

## Mục tiêu

- Luyện khai báo biến đúng cú pháp.
- Chọn kiểu dữ liệu phù hợp với từng tình huống.
- Hiểu khác nhau giữa `int`, `float`, `double`, `decimal`, `bool`, `char`, `string`.
- Biết gán lại giá trị và quan sát sự thay đổi.
- Tập viết code rõ ràng, có tên biến dễ hiểu.
